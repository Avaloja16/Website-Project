Ohjelmointi

Ohjelmoinnissa käytettään useimmiten korkeantason kielijä ohjelmien tekemiseen, kuten Python, C++, Java, Pascal, C# ja PHP. Vaikka tietokoneet toimivat alemman tason kielillä, nämä ovat ihmisille helpompia luettavia. Ohjelma on yksinkertaisuudessaan jono komentoja, jonka tietokone toteuttaa tietyn toiminnan aikaansaamiseksi, ja nämä toiminnot voivat olla matemaattisten laskutoimitusten selvittämistä tai symbolista tekstin läpikäymistä. Ohjelmointikielet eroavat toisistaan hiukan, mutta niiden perustoiminnot ovat lähes samanlaiset; Tehtävälle toiminnolle annetaan tietoa, kerrotaan kuinka sen täytyy kyseinen tieto käsitellä, ja mitä sen täytyy tuottaa.

Ohjelmakielet ovat ns. muodollisia kielijä, ja sen takia niiden ohjeistuksissa täytyy olla erittäin tarkka. Tietokoneet eivät lue tekstiä samalla tavalla kuin me luemme, jonka vuoksi tekemämme koodi ei välttämättä toimi kuin oletamme. Virheiden etsintä on yhtä iso ohjelmointia kuin varsinainen koodaaminen. Erilaisia virheitä ovat syntaksi-, semanttiset-, ja suorituksenaikaiset virheet.

Ohjelmmoinnissa käytetään arvoja, joita ohjelmat pystyvät manipuloimaan toimintojen toteuttamiseen. Arvot jaetaa eri luokkiin kuten integer, string ja float. Luokasta riippuen ne käyttäytyvät eri tavalla, mutta niiden manipulointi tapahtuu samalla tavalla: muuttujilla. Muuttujat voivat olla numeroita tai tekstipätkijä, ja näiden muuttujien arvot määrittää sen mihin luokkaan ne kuuluvat. Arvon lisäksi muuttujilla on aina nimi, ja nimi voi olla sen kaltainen kuin ohjelmoija haluaa sen olevan. Ainoa rajoite nimissä voi olla syntax.

Mainitsemillani arvo-luokilla on rajoituksia toistensa kanssa toimimisessa, mutta onneksi pystymme näitä luokkia muuttamaan tarpeen vaatiessa sopiviksi. Koodi luetaan yleisesti ylhäältä alaspäin, mutta laskutoimituksissa voi tapahtua monta asiaa samaan aikaan yhdellä linjalla. Matematiikan laskutoimitusten järjestyssäännöt pätevät ohjelmoinnissa, joten ongelmia ei synny kunham muistaa PEMDAS:in.

Programming

Ohjelmointikielet

Python

Python on ohjelmointikielenä helppo ja selkeä kieli, ja sitä ehdotetaankin ensimmäiseksi kieleksi, joka opetella. Siinä voidaan tuottaa ja ajaa skriptejä komentoriviltä, toteuttaen laskentaa ja muodostamalla kuvaajia. Se on tulkattava kieli, ja näin ollen sillä luodut skriptit voidaan ajaa välittömästi. Siinä toimii useimmat ohjelmointiparadigmat, joista tyypillisimmät ovat olio, proseduriaalinen ja funktionaalinen.

Pythonissa on dynaaminen tyypitys tarkoittaen sitä, että muuttujaa ei tarvitse erikseen tyypittää. Jos muuttujalle n annetaan arvoksi esim. 15, sen tyypiksi tulee automaattisesti integer. Tämä aiheuttaa kuitenkin ajo-poikkeuksia skriptejä ajatessa, mikäli koodista löytyy tyyppivirheitä. Koodin ajaminen Pythonilla voi olla hidasta, sillä se joutuu ajon aikana kääntämään koodin tavukoodiksi. Prosessia voidaan nopeuttaa tekemällä koodi alusta asti tavukoodilla.

Pythonille on saatavilla käyttöliittymiä tietokantojen käsittelyyn. Viime vuosina python on myös yleistynyt verkkosivujen ohjelmointikielenä ja sitä voidaan ajaa palvelimelta WSGI (Web Server Gateway Interface) – käyttöliittymän avulla, joka parantaa pythonilla tehtyjen verkkosivujen suorituskykyä. Pythonia voidaan laajentaa C -ja C++ kielillä, sekä sitä voidaan myös integroida C -ja C++ kielisiin ohjelmiin skripteinä.

Kaiken tämän lisäksi pythonilla voidaan myös piirtää grafiikka kilpikonna moduulin avulla.

C++

Kielen nimestä voi päätellä, että C++ perustuu C-kieleen, ja se onkin periaatteessa C-kieli lisätyillä ominaisuuksilla. Se sisältää mm. luokat, periytymisen, mallit ja poikkeukset. Tyypillisesti C-kielellä kirjoitetut ohjelmat toimivat myös C++:ssa. Kieli on haastava siinä mielessä, että se odottaa sen käyttäjän tietävän kuinka ohjelmoida, ja kuinka sitä käyttää. Tämän takia pienetkin virheet aiheuttavat ajo-ongelmia. Haasteellisuuden kuitenkin palkitsee se, että käyttäjällä on mahdollisuus suureen hallintaan.

Java

Alun perin Java saavutti suuren suosionsa sen laitteistoriippumattomuuden ansiosta, kuten myös sen C++-kielen samankaltaisuus ja sen mukana tulevan kattavan standardikirjaston takia. Siinä on helpommin opeteltava kielioppi, ja muihin ohjelmointikieliin verrattuna erilainen toteutus, sillä sen koodi suoritetaan erillisessä virtuaalikoneessa. Kieli on siten rajoittunut, ettei se pysty toimimaan virtuaalikoneen ulkopuolisen ympäristön kanssa, sekä että se on hieman hitaampi. Virtuaalikone kuitenkin takaa turvallisen työympäristön.

Java suunniteltiin selaimessa käytettäviä sovelmia varten, mutta kasvoi käytettäväksi palvelimien kanssa, sekä pienlaitteiden käytössä. Tämän takia Javan kehitysympäristön kehittyessä ohjelmointikielestä on julkaistu monta versiota. Java Standard Edition (SE) on perus-versio, Java Enterprise Edition (EE) on suunniteltu palvelin käyttöön ja Java Micro Edition (ME) on tarkoitettu erilaisiin pienlaitteisiin tai laitteisiin, joilla on rajoituksia.

Javaa on mahdollista käyttää Java-ohjelmointikielen ulkopuolella, sillä monet muut ohjelmointikielet sisältävät oman kääntäjän Javan tavukoodia varten. Näihin kieliin kuuluvat mm. Python ja Ruby.
